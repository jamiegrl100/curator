Markdown# Quick StartThis guide gets you from a fresh Curator install to a working digital storefront in ~15 minutes.## 1) Log in to the Admin1. Visit your Curator URL (example: `https://your-domain.com`)2. Log in to the admin panel3. Confirm you can access:   - Products   - Orders   - Downloads / Files   - SettingsIf you don’t have an admin user yet, complete the installer step from [Installation](./INSTALLATION.md).---## 2) Configure the basicsGo to **Admin → Store Settings** and set:- Store name- Support email- Default currency- Legal links (optional but recommended)Go to **Admin → Email (SMTP)** (optional):- Add SMTP settings- Send a test email> SMTP is strongly recommended if you’re selling digital products (customers expect receipts and download emails).---## 3) Create your first product (digital)Go to **Admin → Products → New Product** and fill out:### Basic- Title- Slug (auto is fine)- Short description- Category + tags (optional)### Pricing- Price### Content- Add your product description (WYSIWYG)- Add highlights/features/FAQ (optional but improves conversion)### Media- Upload a product gallery image or screenshots (recommended)### Files- Upload one or more digital files for the product (ZIP, PDF, AppImage, etc.)- Add a label (e.g., “Linux AppImage”, “PDF Guide”, “Source Package”)- Reorder files if neededSave the product.---## 4) Publish it and verify it appears in searchSet product status to **Published/Active**.Now verify:- Your product appears in marketplace/search listings- Clicking the product opens the product page- The product page displays:  - price  - description  - gallery  - highlights/features/FAQ (if added)---## 5) Test the full purchase → download flow### Checkout1. Add the product to cart2. Go to checkout3. Enter the required information (including tax location if enabled)4. Complete payment using your hosted checkout gateway### DownloadsAfter payment:- Confirm an order exists in Admin → Orders- Log in as the customer- Visit **Account → Downloads**- Confirm:  - all purchased files appear  - each file downloads correctly  - “Download all as ZIP” works (if enabled)---## 6) Verify emails (optional but recommended)If SMTP is configured, confirm:- Order confirmation email is sent- Downloads-ready email is sent- Password reset email works---## 7) Install your first theme/plugin### Themes- Download a theme ZIP- Admin → Themes → Upload → Activate### Plugins- Download a plugin ZIP- Admin → Plugins → Upload → Enable> If you run Curator Marketplace, only publish reviewed plugins/themes to keep your ecosystem clean.---## Next: Learn the platform- [Themes](./THEMES.md)- [Plugins](./PLUGINS.md)- [Data Models](./DATA_MODELS.md)Copy
